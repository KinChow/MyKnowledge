# 对象图

对象图为类图的一个实例。

对象图是对象及对象之间关系的集合。

对象图表示系统的静态视图。

对象图的用法类似于类图，但它们用于从实际角度构建系统原型。

对象图派生自类图，因此对象图依赖于类图。

对象图表示类图的一个实例。类图和对象图的基本概念相似。对象图也表示系统的静态视图，但此静态视图是系统在特定时刻的快照。

对象图用于将一组对象及其关系呈现为实例。



## 目的

- 正向和逆向工程。
- 系统的对象关系
- 交互的静态视图。
- 从实际角度理解对象行为及其关系。



## 哪里使用

- 制作系统原型。
- 逆向工程。
- 建模复杂的数据结构。
- 从实际角度理解系统。
